## Что реализовано — по функционалу

### ✅ Публичный сайт

- Главная страница с hero, статистикой, картами проектов
- Страница «О центре»
- Новости: список + фильтр по категориям + пагинация + поиск
- Страница новости с WYSIWYG контентом
- Раздел «Наука»: подразделения, проекты
- База знаний: категории + поиск
- Медиагалерея: фото и видео
- Страница «Документы»: аккордеон по папкам из `public/documents/`
- Страница «Закупки» с ссылками на ЕИС
- Страница «Партнёры»
- Страница «Контакты» с формой (отправка email)
- Страница «Образование» (заглушка / контент в наполнении)

### ✅ CMS-панель (`/admin/`)

12 разделов, весь интерфейс на русском языке:

| Раздел | URL | Функционал |
|---|---|---|
| Новости | `/admin/news` | CRUD + WYSIWYG + кросс-постинг + категории |
| Страницы | `/admin/pages` | CRUD страниц с WYSIWYG |
| База знаний | `/admin/knowledge` | CRUD + категории |
| Медиа | `/admin/media` | Загрузка фото/видео |
| Команда | `/admin/team` | Карточки сотрудников |
| Партнёры | `/admin/partners` | Карточки партнёров |
| Закупки | `/admin/procurements` | CRUD + ссылки ЕИС |
| Проекты | `/admin/projects` | Научные проекты |
| Публикации | `/admin/publications` | Научные публикации |
| Подразделения | `/admin/departments` | Структура организации |
| Документы | `/admin/documents` | (БД-документы) |
| Файловый менеджер | `/admin/files` | Drag & drop загрузка |
| Настройки | `/admin/settings` | Настройки сайта |

### ✅ Кросс-постинг

- Telegram (bot API) — реализован
- ВКонтакте (VK API) — реализован
- MAX — модуль создан, требует API-ключ
- Яндекс.Дзен — модуль создан, требует API-ключ
- Одноклассники — модуль создан, требует API-ключ
- **Retry UI**: при ошибке — кнопка повторной публикации на конкретную площадку
- Журнал публикаций (`crosspost_log`)
- Компонент `CrosspostPanel.tsx` — чекбоксы + статусы + retry

### ✅ SEO

- `robots.ts` — запрещает индексацию `/admin/` и `/api/`
- `sitemap.ts` — статика + все опубликованные новости, revalidate 1 час
- OpenGraph + Twitter Card: глобальные (layout.tsx) + per-article (news/[slug])
- `public/llms.txt` — описание для AI-краулеров (ChatGPT, Perplexity и др.)

### ✅ AI API (`/api/v1/`)

| Endpoint | Описание |
|---|---|
| `GET /api/v1/search` | Полнотекстовый поиск через Meilisearch |
| `GET /api/v1/stats` | Статистика портала (новости, проекты, сотрудники) |
| `GET /api/v1/departments` | Список подразделений |
| `GET /api/v1/knowledge` | База знаний |
| `POST /api/v1/ai/query` | RAG-запрос (инфраструктура подготовлена) |

### ✅ Документы (`/documents`)

Статическая страница, автоматически читает папки из `public/documents/`:

```
public/documents/
  1. Сведения об организации/     (4 PDF)
  2. Государственное задание/     (8 PDF, 2016–2025)
  3. Кадровое обеспечение/        (8 PDF)
```

Добавить новый раздел = добавить папку. Сайт подхватит автоматически.
