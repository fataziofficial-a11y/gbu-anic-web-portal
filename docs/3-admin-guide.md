# Руководство администратора CMS — ГБУ АНИЦ

## Доступ

- URL: `https://your-domain.ru/admin`
- Логин по email и паролю
- Два уровня доступа: **Администратор** (полный) и **Редактор** (без настроек)

---

## Публикация новости — 3 шага

1. **Новости → + Новая новость**
2. Заполнить: заголовок, текст (WYSIWYG), категория, теги
3. Кнопка **«Опубликовать»** (или «Сохранить черновик»)

При публикации можно сразу отправить в Telegram/ВКонтакте — отметить галочки на правой панели.

---

## Разделы CMS

### Новости (`/admin/news`)
- Список новостей с фильтром по статусу
- Статусы: **Черновик** / **Опубликована** / **Архив**
- WYSIWYG-редактор (Tiptap): жирный, курсив, заголовки, списки, цитаты, ссылки
- SEO: заголовок и описание для поисковиков
- Категория и хэш-теги

### Страницы (`/admin/pages`)
- Иерархические страницы сайта
- Шаблоны: `default` / `about` / `contacts`
- Родительская страница (вложенность)

### База знаний (`/admin/knowledge`)
- Научные статьи и методические материалы
- Категория и подразделение
- Теги: вводить через Enter или запятую
- Статусы: Черновик / Опубликован

### Сотрудники (`/admin/team`)
- ФИО, должность, подразделение
- Фото (выбор из загруженных файлов)
- Email, краткая биография
- Порядок сортировки

### Подразделения (`/admin/departments`)
- Управление прямо на странице (без перехода)
- Название, slug, описание, руководитель

### Проекты (`/admin/projects`)
- Статус: Активный / Планируется / Завершён
- Даты начала и окончания
- Привязка к подразделению

### Публикации (`/admin/publications`)
- Научные публикации с DOI
- Авторы, журнал, год
- Привязка к подразделению

### Файлы (`/admin/files`)
- Drag & drop загрузка
- Папки: **Медиа** (фото), **Документы**, **База знаний**
- Просмотр миниатюр и размеров

### Кросс-постинг (`/admin/crosspost`)
- История публикаций в соцсети
- Платформа, статус (✅/❌), ссылка

### Настройки (`/admin/settings`)
- Название и описание сайта
- Контакты (email, телефон, адрес)
- Ссылки на соцсети (Telegram, VK, ОК, MAX)
- robots.txt

## Редактор текста (WYSIWYG)

| Кнопка | Горячая клавиша | Действие |
|--------|----------------|----------|
| **Ж** | Ctrl+B | Жирный |
| **К** | Ctrl+I | Курсив |
| ~~З~~ | — | Зачёркнутый |
| H2 | — | Заголовок 2-го уровня |
| H3 | — | Заголовок 3-го уровня |
| ● | — | Маркированный список |
| 1. | — | Нумерованный список |
| " | — | Цитата |
| ↩ | Ctrl+Z | Отменить |
| ↪ | Ctrl+Y | Повторить |

---

## Кросс-постинг в соцсети

При создании/редактировании новости — правая панель «Кросс-постинг»:
1. Отметить галочки нужных платформ (Telegram, ВКонтакте)
2. Нажать **«Опубликовать»**
3. Проверить результат в разделе **Кросс-постинг**

Токены платформ настраиваются в **Настройки** (для администраторов).

## Часто задаваемые вопросы

**Q: Как восстановить удалённую новость?**
A: Удаление необратимо. Используйте черновики вместо удаления.

**Q: Можно ли публиковать одновременно в несколько соцсетей?**
A: Да, отметьте все нужные галочки перед публикацией.

**Q: Где хранятся загруженные изображения?**
A: В папке `/uploads/media/` на сервере. В файловом менеджере можно просмотреть все файлы.

**Q: Как изменить порядок сотрудников?**
A: Поле «Порядок сортировки» — меньшее число выводится первым.
