# База знаний — структура и управление

## Что такое база знаний

Раздел сайта для публикации научных статей, методических материалов, отчётов и прочих документов Арктического НИЦ. Материалы доступны публично на `/knowledge-base`, управляются через CMS.

---

## Структура

```
Категории (kb_categories)
  └── Материалы (knowledge_items)
        ├── Текст (Tiptap JSONB)
        ├── Теги []
        ├── Подразделение
        └── Автор
```

### Категории

Иерархические (parent_id). Управление: `/admin/knowledge` → через API (пока без отдельного экрана категорий в CMS).

Примеры категорий для АНИЦ:
- Климатология
- Экология Арктики
- Биология северных экосистем
- Социально-экономические исследования
- Методические материалы
- Отчёты о НИР

### Материалы

| Поле | Описание |
|------|----------|
| title | Заголовок |
| slug | URL-идентификатор (автогенерация) |
| content | Текст в формате Tiptap JSON |
| categoryId | Категория |
| departmentId | Подразделение |
| tags | Массив тегов |
| status | draft / published |
| publishedAt | Дата публикации |

---

## Добавление материала

1. `/admin/knowledge` → **+ Новый материал**
2. Заголовок → slug генерируется автоматически
3. Категория и подразделение (опционально)
4. Теги: вводить через Enter или запятую
5. Текст в редакторе Tiptap
6. **«Опубликовать»**

---

## Массовый импорт

Скрипт импорта из PDF/DOCX будет добавлен в следующей фазе.

Пока — ручное добавление через CMS или прямой INSERT в БД:

```sql
INSERT INTO knowledge_items (title, slug, content, status, published_at, category_id)
VALUES (
  'Название статьи',
  'nazvanie-stati',
  '{"type":"doc","content":[{"type":"paragraph","content":[{"type":"text","text":"Текст статьи"}]}]}',
  'published',
  NOW(),
  1
);
```
